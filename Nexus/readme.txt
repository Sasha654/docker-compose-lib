Сборка c Nexus OSS, перед которым стоит nginx и отдаёт наружу http и https.

Основные файлы:
 - localhost.crt - открытый ключ на сайте nginx
 - localhost.key - приватный ключ там же
 - nginx.conf - конфигурация проксирующего сайта nginx

WARN!
Для успешной проверки доверия к https в Windows, сертификат (crt) необходимо добавить в раздел доверенных _издателей_ ОС (Root CA)

Дополнительно:
Nexus:
 - админские реквизиты по умолчанию: admin/admin
Nginx:
 - слушает на портах 9000/http и 9001/https
